Задание 2
1.Создаем консольное приложение, в котором будет происходить сортировка списка фамилий из пяти человек.
Сортировка будет происходить при помощи события. 
Сортировка происходит при введении пользователем либо числа 1 (сортировка А-Я), либо числа 2 (сортировка Я-А) или число 3 (выход).

2.Дополнительно реализуем проверку введённых данных от пользователя через конструкцию TryCatchFinally с использованием собственного типа исключения.
